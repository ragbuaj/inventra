DejaVu Fonts (DejaVuSansCondensed family)

These TrueType font files (DejaVuSansCondensed.ttf, DejaVuSansCondensed-Bold.ttf,
DejaVuSansCondensed-Oblique.ttf, DejaVuSansCondensed-BoldOblique.ttf) are part of the
DejaVu Fonts project (https://dejavu-fonts.github.io/), which is a public, permissively
licensed font family derived from the Bitstream Vera Fonts. DejaVu fonts are released
under a license that permits free use, modification, and redistribution (including for
commercial purposes), derived from the original Bitstream Vera Fonts Copyright and
license terms.

These specific font files were sourced from the Go module
github.com/go-pdf/fpdf@v0.9.0 (its bundled `font/` directory), which itself vendors
these fonts for use with the fpdf PDF generation library. They are included here
unmodified for embedding Unicode text (UTF-8) into PDFs generated by this project.

See the DejaVu Fonts project and the Bitstream Vera Fonts license for full license text:
https://dejavu-fonts.github.io/License.html
